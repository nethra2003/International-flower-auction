DBMS Project

The International Flower Auction is a web-based application that simulates real-world flower trading across global markets. The platform is designed to connect flower growers, buyers, and auctioneers, allowing them to participate in a transparent and efficient auction process.

Users can view detailed flower listings, place bids in real time, and track price changes for different flower varieties. The system focuses on improving traditional agricultural trading by using technology to enhance user interaction, data management, and overall efficiency.

This project highlights practical implementation of database concepts along with frontend and backend integration, making it a strong example of how web technologies can modernize conventional market systems.

Technologies Used:
HTML, CSS, PHP, MySQL
